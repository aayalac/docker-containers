# Comandos para administrar registros de Docker

Se pueden ejecutar desde el CLI:

Uso: **[docker][registry][comando]**

- Permite listar los eventos del registro.
     ```bash
    docker registry events
    ```
- Permite visualizar el historial de impagenes del registro.
     ```bash
    docker registry history
    ```
- Permite mostrar información de un registro.
     ```bash
    docker registry info
    ```
- Permite inspeccionar imágenes del registro.
     ```bash
    docker registry inspect
    ```
- Permite mostrar infomración de la lista de trabajos del registro.
     ```bash
    docker registry joblogs
    ```
- Permite listar los trabajos del registro.
     ```bash
    docker registry jobs
    ```
- Permite listar las imágenes del registro.
     ```bash
    docker registry ls
    ```
- Permite eliminar una imagen del registro.
     ```bash
    docker registry rmi
    ```
