# Comandos para administrar volumenes en Docker

Se pueden ejecutar desde el CLI:

Uso: **[docker][volume][comando]**

- Permite crear un volumen.
     ```bash
    docker volume create
    ```
- Permite mostrar información detallada de los volúmenes.
     ```bash
    docker volume inspect
    ```
- Permite listar los volúmenes.
     ```bash
    docker volume ls
    ```
- Permite eliminar todos los volúmenes no utilizados.
     ```bash
    docker volume prune
    ```
- Permite eliminar volúmenes.
     ```bash
    docker volume rm
    ```
